Introduction

Hello students!


This website was designed as a way for young learners to practice their fundamental chemistry. Below you be able to find both the game to practice with and some instructions on how to play. Make sure to check out the rules before diving in, and have fun!

This browser does not support canvas

How To Play

The main goal of the game is fairly simply to understand. Upon starting, you will be asked to form a number of a specific molecule. This can be done by firing orbs that represent some common elements. Until you have formed the compound listed, any other combinations of elements will not result in a molecule forming.


The game will end if you fill the screen with no room to make new elements.
